# ez-openssl

OpenSSL built from source, git tag openssl-3.3.0, with the following configure options
```bash
Configure \
  no-shared \
  no-stdio \
  no-apps \
  no-deprecated \
  no-autoload-config \
  no-tests \
  threads \
  no-default-thread-pool \
  no-thread-pool \
  enable-engine \
  no-legacy
```
